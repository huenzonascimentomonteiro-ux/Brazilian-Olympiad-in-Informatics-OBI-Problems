Cibele, Camila, and Celeste are three inseparable sisters. They are always together and love playing sports, reading, cooking, playing computer games... Now, they are learning computer programming to develop their own games.

However, none of that matters for this task: we are interested only in their ages. We know that Cibele was born before Camila, and Celeste was born after Camila.

Given three integers representing the sisters' ages, write a program to determine Camila's age.

Input
The input consists of three lines, each containing an integer N representing the age of one of the sisters.

Output
Your program must produce a single line containing a single integer: Camila's age.
